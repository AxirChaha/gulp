# GULP шаблон
### Используемые пакеты -
- gulp
- browser-sync
- del
- gulp-file-include
- gulp-htmlmin
- gulp-imagemin
- gulp-sass
- gulp-uglify-es

> Запуск для работы - ```gulp```

> Создание готовой (минифицированной) версии - ```gulp build```

> Избыточные папки при разработке только для удобства и отсутствия ошибок IDE!!!

>Переименование минифицированных файлов не происходит, ибо нафига ;)
